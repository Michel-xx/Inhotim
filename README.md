# Inhotim

Este projeto é um site estático em HTML e CSS sobre o Instituto Inhotim, em Brumadinho, Minas Gerais. O conteúdo apresenta a história do local, a origem do nome "Inhotim", a transformação de uma fazenda em um importante centro de arte contemporânea e botânica, além de informações sobre o espaço, o acervo e a relevância cultural do instituto.

## O que o site mostra

- A história e a origem do nome do Inhotim
- A evolução do local desde uma fazenda até um museu a céu aberto
- Informações sobre a relação entre arte, paisagismo e preservação ambiental
- Dados sobre o espaço, o acervo artístico e o jardim botânico

## Tecnologias utilizadas

- HTML
- CSS
